About Zuma:

Background and Overview

  • **Zuma** is a classic game where user has to make sure not to let the ball chain reach skull emblem in order to win.

  • Checkout the rules on the right.

  • Give it a try, it's fun!

  • Feel free to come back to the about section any time. It will pause your game automatically. ;)

Architecture and Technologies

This project was implemented using following technologies:

  • Vanilla JavaScript for DOM manipulation, game structure and game logic.
  • HTML5 Canvas for DOM manipulation and rendering.
  • Webpack to bundle and serve up the various scripts.
  • Howler audio library was used for sound generation, processing and control.
